Our Project

We developed a web-based Attendance Tracker to help students monitor their attendance throughout the semester in a simple and interactive way. Users can:

  • Add or customize subjects
  • Mark attendance daily
  • View attendance percentage through graphs
  • Reset attendance for a new semester
  • Login securely using AWS Cognito

This project is built using HTML, CSS, JavaScript, PHP, and SQLite, with integration to AWS and COGNITO. It is designed to be student-friendly, fast, and visually appealing.
